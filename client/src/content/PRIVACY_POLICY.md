**PRIVACY POLICY**

RepoMax

*Last Updated: February 13, 2026*

1\. Introduction

Welcome to ReadyRepo. We are committed to protecting your privacy and
ensuring the security of your personal information. This Privacy Policy
explains how we collect, use, disclose, and safeguard your data when you
use our web application and API services.

RepoMax is a service that helps you optimize your GitHub presence for
specific job applications by analyzing job descriptions and generating
tailored README files based on your repository metadata.

By using ReadyRepo, you agree to the collection and use of information in
accordance with this Privacy Policy. If you do not agree with our
policies and practices, please do not use our services.

2\. Information We Collect

2.1 Information You Provide

**Account Information:**

-   Email address and authentication credentials

-   GitHub account information (username, profile data)

-   GitHub OAuth access tokens (encrypted and stored securely)

**User-Generated Content:**

-   Job descriptions you input for analysis

-   Repository URLs and metadata

-   README content and customization preferences

2.2 Automatically Collected Information

**Usage Data:**

-   Pages visited and features used

-   Time and date of visits

-   Time spent on pages

-   Error logs and diagnostic data

**Device and Technical Information:**

-   IP address

-   Browser type and version

-   Operating system

-   Device identifiers

-   Referral source

2.3 Information from Third-Party Services

We receive information from third-party services you connect with:

-   **GitHub API:** Repository data, commit history, README content,
    repository metadata, and public profile information

-   **Google Analytics:** Anonymous usage statistics and behavioral data

We use the collected information for the following purposes:

**Service Provision:**

-   To provide, operate, and maintain RepoMax services

-   To process job descriptions and generate tone analyses

-   To create AI-assisted README rewrites tailored to job postings

-   To access and retrieve your GitHub repository data

**Service Improvement:**

-   To understand and analyze how you use our services

-   To develop new features and functionality

-   To improve user experience and interface design

**Communication:**

-   To send you technical notifications and updates

-   To respond to your inquiries and support requests

-   To notify you about changes to our services or policies

**Security and Legal Compliance:**

-   To detect, prevent, and address technical issues

-   To protect against fraudulent or illegal activity

-   To comply with legal obligations and enforce our terms

4\. Third-Party Services and Data Sharing

We use the following third-party services to operate RepoMax:

We use Google Gemini to power our tone analysis and README generation

-   Your job descriptions and repository metadata are sent to Google's

-   Google processes this data to generate analyses and content
    suggestions

-   Google's privacy policy and terms of service apply to this
    processing

4.2 Firebase and Google Cloud

We use Firebase and Google Cloud Platform for:

-   User authentication and account management

-   Database storage and management

-   Application hosting and infrastructure

-   Server operations (EU-Madrid region)

4.3 Google Analytics

We use Google Analytics to understand how users interact with our
service. This includes:

-   Anonymous usage statistics

-   Page views and user flows

-   Demographic and interest data (when available)

4.4 GitHub API

We integrate with GitHub's API to:

-   Retrieve your repository information and metadata

-   Access README files for analysis and rewriting

-   Authenticate your account

We only request the minimum GitHub permissions necessary to provide our
services. Your GitHub data is subject to GitHub's privacy policy.

4.5 No Sale of Personal Data

**We do not sell, trade, or rent your personal information to third
parties.**

5\. Data Storage and Security

5.1 Data Location

Spain), providing enhanced data protection under GDPR regulations.

5.2 Security Measures

We implement industry-standard security measures to protect your
information:

-   Encryption of data in transit using HTTPS/TLS

-   Encryption of sensitive data at rest

-   Secure authentication via Firebase Authentication

-   Regular security assessments and updates

-   Access controls and monitoring

-   Secure OAuth token storage

5.3 Data Retention

We retain your personal information only for as long as necessary to:

-   Provide you with our services

-   Comply with legal obligations

-   Resolve disputes and enforce our agreements

When you delete your account, we will delete or anonymize your personal
information within 30 days, except where we are required to retain it
